# CWE-414 Guidance

## CWE ID

CWE-414

## CWE Name

Missing Lock Check

## Source

MITRE CWE official entry:

https://cwe.mitre.org/data/definitions/414.html

## Weakness Summary

A product does not check to see if a lock is present before performing sensitive operations on a resource.

## Mitigation Guidance

- Phase: Architecture and Design - Implement a reliable lock mechanism.

## Dataset Usage

This guidance will be used when generating secure counterparts for vulnerable samples mapped to CWE-414.

The secure counterpart should preserve intended functionality while removing or mitigating the target weakness using CWE-specific guidance.
