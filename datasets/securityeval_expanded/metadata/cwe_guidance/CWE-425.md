# CWE-425 Guidance

## CWE ID

CWE-425

## CWE Name

Direct Request ('Forced Browsing')

## Source

MITRE CWE official entry:

https://cwe.mitre.org/data/definitions/425.html

## Weakness Summary

The web application does not adequately enforce appropriate authorization on all restricted URLs, scripts, or files.

## Mitigation Guidance

- Phase: Architecture and Design - Apply appropriate access control authorizations for each access to all restricted URLs, scripts or files.
- Phase: Architecture and Design - Consider using MVC based frameworks such as Struts.

## Dataset Usage

This guidance will be used when generating secure counterparts for vulnerable samples mapped to CWE-425.

The secure counterpart should preserve intended functionality while removing or mitigating the target weakness using CWE-specific guidance.
