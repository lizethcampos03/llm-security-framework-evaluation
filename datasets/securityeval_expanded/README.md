# SecurityEval Expanded Dataset

This directory contains the expanded contextual benchmark used for framework evaluation.

The dataset is derived from SecurityEval methodology and extends the benchmark with:

- Vulnerable implementations
- Secure counterparts
- Security context profiles
- Metadata records
- Validation notes

## Directory Structure

```text
vulnerable/
secure/
context_profiles/
labels/
metadata/
validation/
```

## Purpose

The dataset supports:

- Experiment 1
- Experiment 2
- Experiment 3

and serves as the primary evaluation benchmark for the framework.
